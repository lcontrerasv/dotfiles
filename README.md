lcontrerasv dotfiles
===========================

Requirements
------------

Set zsh as your login shell:

	chsh -s $(which zsh)

Install
------------

1.- Clone on your computer:

	git clone --recurse-submodules https://github.com/lcontrerasv/dotfiles.git

2.- Run the setup:

	.../dotfiles/setup -i

Update
------------

1.- Run the setup for update:

	.../dotfiles/setup -u
